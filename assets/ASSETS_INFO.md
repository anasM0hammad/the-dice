# Splash Screen & Icon Assets

## Current Assets:
- `splash.png` - Splash screen with dice icon (1284x2778 px)
- `icon.png` - App icon with dice (1024x1024 px)

## How to Replace:

1. **Splash Screen** (`splash.png`):
   - Size: 1284x2778 pixels (iPhone size)
   - Background: Dark theme (#1a1a2e)
   - Place your custom splash image here

2. **App Icon** (`icon.png`):
   - Size: 1024x1024 pixels
   - Should be a simple, recognizable icon
   - Place your custom icon here

3. After replacing, run:
   ```bash
   npx expo prebuild --clean
   ```

## Tips:
- Keep splash screen simple with centered logo
- Icon should work at small sizes (avoid too much detail)
- Use tools like Figma or Canva to create professional assets
